# Hacker Speak Telegram Bot 

Простой Telegram-бот на **Python**, который переводит обычный текст на "хакерский язык" (Leet Speak).

## Технологии:
* **Python 3.13**
* **pyTelegramBotAPI** (библиотека для бота)
* **python-dotenv** (безопасное хранение токена)

## Как запустить:
1. Клонируйте репозиторий.
2. Создайте файл `.env` в корневой папке.
3. Добавьте в него ваш токен: `BOT_TOKEN=ваш_токен`.
4. Запустите `main.py`.
